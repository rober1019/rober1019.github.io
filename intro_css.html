Selección por nombre

Se puede estilizar poniendo el nombre de la etiqueta.

Las etiquetas h2 estan todas estilizadas de esa forma poniendo el nombre de la etiqueta que se quiere estilizar.

Selección por id

Se puede estilizar poniendo un id específico a cada etiqueta.

Estas dos lineas estan estilizadas con id y "#" seguido del id asignado.

Selección por clase

Se puede estilizar asignando una clase a cada elemento que quieras estilizar sin tener que poner un id diferente a cada uno de los elementos que quieras estilizar.

Estas lineas están estilizadas con una clase usando la etiqueta "class" y usando "." seguido de la clase asignada.
