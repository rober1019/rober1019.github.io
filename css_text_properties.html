Propiedades de texto CSS Roberto

Fuente de texto genéricas

Puedes cambiar la fuente del texto usando la propiedad "font-family" y escribiendo una familia de fuente genérica.

Este texto está escrito con la fuente "serif".

Fuente de texto específica

Puedes cambiar la fuente del texto por una fuente específica usando la propiedad "font-family" y poniendo el nombre de la fuente específica entre comillas. Por ejemplo: font-family:"arial";

Este texto está escrito con la fuente "arial".

Tamaño de fuente en píxeles

Puedes cambiar el tamaño de la letra utilizando el aributo "font-size" y escribiendo el número de píxeles del que quieres que sea el tamaño de la letra.

Este texto está escrito con una letra de 17px de tamaño.

Tamaño de fuente (em)

Puedes cambiar el tamaño de la letra con una proporción usando la propiedad "font-size" y después poniendo la proporción que quieres acompañado de "em".

Este texto está escrito 1.2 veces mayor que el resto.

Estilo de fuente

Podemos cambiar el estilo de fuente con la propiedad "font-style" y poniendo el estio con el que quieres modificar el texto.

Este texto esta escrito con "italic".

Altura de lineas

Puedes cabiar la distancia de las lineas del texto con la propiedad "line-height".

Este texto está escrito a 20 píxeles de las demás lineas.

Alinear texto

Puedes alinear texto usando la propiedad "text-align".

Este texto está alineado en el centro.

Decoración del texto

Puedes decorar el texto usando la propiedad "text-decoration", puedes por ejemplo quitar el subrayado de enlaces o subrayar texto.

Este texto está subrayado con esta propiedad
