Span

Puedes usar la etiqueta "span" para seleccionar texto que este en linea.
Este texto esta seleccionado y estilizado con "span".

Div

Puedes seleccionar elementos que no esten en la misma linea y agruparlos todos en un mismo bloque con la etiqueta "div".
Estas dos lineas estan seleccionadas con "div".

Overflow

Dependiendo de la anchura o la altura a la que queramos ajustar un bloque podemos evitar que se desborde el texto usando la etiqueta "overflow" y se añaden barras de desplazamiento a este.
Este bloque está adaptado con "overflow"

Box model

El modelo de caja es un módulo CSS que define cajas rectangulares, incluyendo sus rellenos y márgenes, que son generadas para los elementos que incluímos en una página web.
